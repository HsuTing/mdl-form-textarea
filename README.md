# Textarea for mdl-form

## Parameter

- `style` -> change style
- `label` -> name of textarea
- `rows` -> rows of textarea
- `change`(function) -> you can do something when teatarea is changed. `1st` argument is this component value.

## Example

```
{   
  'id': 'component_id',
  'label': 'Label',
  'style': {'color': 'red'},
  'rows': '10'
}
```
